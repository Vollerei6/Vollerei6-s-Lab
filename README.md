# 绿神话-吗喽

## 简介

《绿神话-吗喽》是由[Vollerei6](https://github.com/Vollerei6)不完全自主研发的一款全新不开放世界冒险游戏。游戏发生在一个被称作「命令行」的幻想世界，在这里，被神人选中的人将被授予「普通攻击」，引导扣一之力。你将扮演一位名为「player」的神秘角色，在自由的旅行中不邂逅性格各异、能力独特的同伴们，和他们一起击败强敌，找回失散的亲人；同时，逐步发掘「奶龙」的真相。

## 编译

```shell
mkdir build
cd build
cmake ..
make
```

## 运行

```shell
./Vollerei6sLab
```
